# slack-weather-message

슬랙 날씨 요정 프로그램

---

## Installation

해당 프로그램은 Makefile을 통해 간편하게 설치할 수 있습니다.

```bash
  make install
```

Make가 없는 경우, 별도로 설치가 필요합니다. 
- [Make.exe install](https://gnuwin32.sourceforge.net/packages/make.htm)

---

## How to Run

다음 기능을 통해 실행이 가능합니다.

```bash
  make run
```
